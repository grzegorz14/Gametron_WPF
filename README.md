# Gametron_WPF

Desktop application which uses API collecting data from BoardGameGeek profiles. App shows all of the boardgames owned by a user. 
You may search a game by title or sort a collection by some parameter. 

Plans for next features: 
* rate owned board games
* add gameplays with scores
* create settings panel (being able to switch user profile)
